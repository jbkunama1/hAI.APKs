# hAI.APKs

> Eine Sammlung von APKs und APK-bezogenen Ressourcen von **trt/hAI**

---

## 📖 Über dieses Repository

**hAI.APKs** ist das zentrale Archiv für Android-Pakete (APKs) und zugehörige Materialien, die im Rahmen des **trt/hAI**-Projekts erstellt oder kuratiert wurden. Hier findest du fertige APK-Dateien, Build-Ressourcen und weitere APK-bezogene Inhalte.

---

## 📂 Inhalt

| Kategorie | Beschreibung |
|-----------|--------------|
| APKs | Fertige Android-Paketdateien (`.apk`) |
| APK-Ressourcen | Build-Skripte, Manifeste und andere zugehörige Dateien |

---

## 🚀 Installation / Verwendung

1. **APK herunterladen** – Lade die gewünschte `.apk`-Datei aus diesem Repository herunter.
2. **Installation auf Android** – Aktiviere auf deinem Gerät unter *Einstellungen → Sicherheit* die Option **„Unbekannte Quellen"** (bzw. *„Apps aus unbekannten Quellen installieren"*).
3. **APK öffnen** – Öffne die heruntergeladene Datei und folge den Installationsanweisungen.

> ⚠️ **Hinweis:** Installiere APKs nur aus vertrauenswürdigen Quellen. Diese APKs werden von trt/hAI bereitgestellt und sind zur eigenen Nutzung gedacht.

---

## 🛠️ Anforderungen

- Android-Gerät oder Emulator (Android 5.0+)
- Aktivierte Installation aus unbekannten Quellen
